# Todo Webapp

## Introduction

This documentation provides a detailed guide on how to set up and run the Todo Webapp created for the Wisata App Junior Software Engineer Technical Test. The application allows users to manage their tasks efficiently with features such as adding, editing, deleting, updating status, sorting, and filtering todos. Additionally, the app supports both light and dark modes for a better user experience. All todos are store in local storage browser.

## Features

1. Add Todos: Users can add new todos with a title, description, date, time, and place.
2. Edit Todos: Users can edit existing todos.
3. Delete Todos: Users can delete todos.
4. Update Status of Todos: Users can mark todos as completed or pending.
5. Sort Todos: Users can sort todos by title or date.
6. Filter Todos: Users can filter todos based on their status (all, completed, pending).
7. Light and Dark Mode: Users can switch between light and dark mode.

## Installation and Setup

### Prerequisites

1. Node.js (v12.x or later)
2. npm

### Steps to Run the Application Locally

1. Clone the Repository

```sh
git clone [repository-url]
cd [repository-directory]
```

2. Install Dependencies

Using npm:

```sh
npm install
```

3. Run the Development Server

```sh
npm run dev
```

4. Access the Application
   Open your web browser and navigate to [localhost port 3000](http://localhost:3000)
